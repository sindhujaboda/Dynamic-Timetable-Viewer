About the Project

The College Class Schedule Management System is a web-based platform designed to help students, faculty, and staff view and manage class schedules at the college. The system allows users to filter class schedules by parameters such as year, department, section, and day, offering a personalized view of the academic schedule.

Key Features:

  • Interactive Calendar: Easily navigate through months to find class schedules for specific days.
  • Class Schedule Filters: Filter schedules by year, department, section, and day for a customized view.
  • Dynamic Schedule Table: View detailed class schedules including timing, subject, faculty, room number, and more.
  • Resizable and Sliding Table: Resize the schedule table to fit your screen and preferences.
  • Responsive Design: The system is fully responsive, ensuring a smooth experience on any device.

How It Works:

Once logged into the system, users can select their preferred parameters such as year, department, section, and day using a form. The system then displays a table with the relevant schedule information. Users can also view the class schedule on a month-by-month basis using the interactive calendar feature.

Technologies Used:

The project is built using a combination of modern web technologies:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: MySQL
  • Middleware: Body-Parser, CORS

Why Use This System?

This system offers a streamlined and efficient way for students, faculty, and staff to manage their academic schedules. By providing easy-to-navigate features and customizable filtering, users can access the information they need quickly and accurately. Whether you're a student trying to find your classes for the week or a faculty member needing to manage multiple sections, this system simplifies the process.

Future Enhancements:

While the current version of the system is fully functional, future updates will include user authentication, notification features for schedule changes, and the ability to export schedules to PDF or other formats for convenience.
